# Mutation Testing Results

This directory contains mutation testing artifacts generated by Cosmic Ray.

## Files

- `cosmic-ray-summary.txt`: High-level summary of mutation test run (total/killed/survived/unknown)
- `cosmic-ray-survivors.json`: Detailed JSON report of surviving mutations (mutations not killed by tests)
- `cosmic-ray-report.json`: Full Cosmic Ray report with all mutation results
- `cosmic_ray_summary_run.txt`: Console output from running the summarize tool
- `cosmic-ray-dump.err`: Error/stderr output from mutation runs
- `cosmic_ray_full_run.log`: Full execution log from mutation test run
- `cosmic_ray_exec.log`: Log from the cosmic-ray exec command (when available)

## Mutation Testing Workflow

1. **Initialize session**: `cosmic-ray init cosmic-ray.toml cosmic-ray-session.sqlite`
2. **Execute mutations**: `cosmic-ray exec cosmic-ray.toml cosmic-ray-session.sqlite`
3. **Generate report**: `cr-report cosmic-ray-session.sqlite --show-output > cosmic-ray-report.json`
4. **Summarize survivors**: `python tools/cosmic_ray_summarize.py cosmic-ray-survivors.json`
